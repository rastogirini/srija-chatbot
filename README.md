# 🍽️ Srija Chatbot

An AI-powered restaurant reservation chatbot built using Flask, React, and DuckDB.

---

## 🚀 Features

- Restaurant table reservation chatbot
- React frontend UI
- Flask backend API
- DuckDB database integration
- Reservation storage
- Dynamic reservation IDs
- Responsive chatbot interface

---

## 🛠️ Technologies Used

### Frontend
- React
- Vite
- JavaScript
- CSS

### Backend
- Python
- Flask
- DuckDB
